# Profile Picture Instructions

## How to Add Your Profile Picture:

1. **Save your profile picture** as `profile-pic.jpg` in this folder
2. **Recommended dimensions**: 600x600 pixels or larger (square aspect ratio)
3. **File format**: JPG, JPEG, or PNG
4. **File name**: Must be exactly `profile-pic.jpg`

## Current Setup:
- Your portfolio is configured to display the image at 300x300 pixels on desktop
- Responsive design: 250x250 on tablets, 200x200 on mobile
- Circular design with white border and shadow effects
- Hover effects included

## Alternative File Names:
If you prefer a different filename, update the `src` attribute in `views/index.ejs`:
```html
<img src="/images/YOUR_FILENAME.jpg" alt="Abdul Ahad - Full Stack Developer" class="profile-image">
```

## Image Requirements:
- High quality (at least 300x300 pixels)
- Professional appearance
- Good lighting and clear face visibility
- Square or 1:1 aspect ratio recommended
