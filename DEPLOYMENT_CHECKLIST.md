# 🚀 Vercel Deployment Checklist

## ✅ Pre-Deployment Checklist

- [ ] All files are committed to Git
- [ ] Profile picture added to `public/images/profile-pic.jpg`
- [ ] Resume PDF added to `public/documents/resume.pdf`
- [ ] `vercel.json` file created
- [ ] `package.json` has build script
- [ ] Local testing completed (`npm run dev`)

## 🔧 Deployment Steps

### 1. GitHub Setup
- [ ] Create GitHub repository
- [ ] Push code to GitHub
- [ ] Verify all files are uploaded

### 2. Vercel Setup
- [ ] Sign up/Login to Vercel
- [ ] Connect GitHub account
- [ ] Import portfolio repository
- [ ] Configure project settings
- [ ] Deploy project

### 3. Post-Deployment
- [ ] Test all sections work
- [ ] Verify profile picture displays
- [ ] Test resume download
- [ ] Check contact form
- [ ] Test on mobile devices
- [ ] Share portfolio URL

## 📱 Quick Commands

```bash
# Initialize Git (if not done)
git init
git add .
git commit -m "Initial portfolio commit"

# Add GitHub remote
git remote add origin https://github.com/YOUR_USERNAME/portfolio.git
git branch -M main
git push -u origin main

# Update files later
git add .
git commit -m "Update portfolio"
git push
```
